# Accord Strategy & Operations Explorer

Interactive companion to the Hope Rises International segment of the Accord Network AI in Practice webinar (18 May 2026).

The framework is Claire Brosius's; this is an interactive version with HRI examples and concrete first steps for each spoke.

**Live site:** https://hope-rises-international.github.io/accord-strategy-explorer/

## What's inside

`index.html` — a single-file, self-contained web app. Click any spoke to see what HRI has built or is working on in that area, and the first step you could take in your own organization.

The Join button links to the Accord Strategy & Execution Community Group subscription form.
